## Usage

> 页面需在koa-grace下运行，所以请先安装koa-grace，很简单的~ [koa-grace是什么，如何使用？点我查看](https://github.com/xiongwilee/koa-grace#二快速开始)
 
```node
// ~/fe为koa-grace根目录
cd ~/fe/app/

git clone git@github.com:Thunf/aliAR.git aliar
cd aliar/

npm i 

npm run dev

```
